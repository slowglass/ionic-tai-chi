Tai Chi

Welcome to the 'Tai Chi' App. The aim of this app is to help you with your Tai Chi Practice. It provides the following;

Timers
A timer that will run multiple times with optional setup time at start and between each iteration.
Level Monitor
Monitors how level your hands are. Helps you to improve exersises like Swimming Dragon
Notes
Place to store you own notes
Principles
Tai Chi Principles
